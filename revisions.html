My Guidance

Revisions

My revisions to this site were a result of the combined advice from two individuals:

Chatoyre, a video editor/graphic designer and Valentina, a television program producer.

"My Story" Revisions

Chatoyre's layout suggestions:

  • Choose a color palette
  • Left justify all paragraphs
  • Add space between paragraphs
  • Navigation bar is fine at the bottom, but try placing it at the top

Valentina's layout suggestions:

  • Choose a color palette based on header picture, with one accent color
  • Change name color
  • Make sure boxes are the same length
  • Add space between line break and paragraph

"My Passion" Revisions

Chatoyre's layout suggestions:

  • Choose a color palette
  • Left justify all paragraphs
  • Need to have more space between pictures and wording
  • Page too plain and needs more dimension

Valentina's layout suggestions:

  • Keep same color and theme throughout all pages
  • Intro font size too small, needs to be more defined
  • Keep same header image on all pages
  • Make sure all elements are centered

After taking all of the suggestions into consideration and spending time re-evaluating my design choices, I finally have a finished product!
